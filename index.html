Database Design process
What kid of thing are we storing?                         → We are storing a list of cities → We should create a table called 'cities'
What properties does this thing have? → Each city has a name, country, population, and area → The table should have columns of name, country, population, area
What type of data does each of those properties contain? →
name → string →
country → string
population → number
area → number
Each column should indicate the type of data that it is going to store
